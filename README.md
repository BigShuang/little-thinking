# little-thinking
### 闲暇的思考

大家好，我是大爽，自认为是一名教育工作者。

实际生活中，比起技能或者知识上的缺乏或不足，**关于人生发展的困惑**，比如个人的未来方向性的规划（再具体一点比如未来的职业规划），可能是更多人更为头疼的问题。

这里记录并展示一些我关于这一问题的一些具体性的思考，
这些思考是有局限性的，所以：
- 1 欢迎批判交流讨论与补充
- 2 不希望你信我所说的，希望你可以思考我所说的，然后结合自身的情况，思考出自己的结论。
